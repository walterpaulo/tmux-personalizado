## Personalizar o Shell/Bash com Tmux

* tmux;
* Debian e derivados;

```
git clone https://github.com/walterpaulo/tmux-personalizado.git
cd tmux-personalizado
./install.sh
```

[![Barra de status](./images/barra-status.png)](https://youtu.be/1GJq5QJzjuM)

```
tmux
```
Pronto!
